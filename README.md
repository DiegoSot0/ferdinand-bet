# ⚽ Ferdinand Bet

Ferdinand Bet es una plataforma web para administrar una polla deportiva de forma sencilla y moderna.

## Características

- 🏆 Ranking automático de participantes
- ⚽ Pronósticos por partido
- 📊 Cálculo automático de puntajes
- 📱 Diseño responsive
- 📋 Modal con detalle de apuestas
- 🔄 Actualización de resultados mediante JSON
- 🌐 Compatible con GitHub Pages
- 🎨 Diseño inspirado en aplicaciones deportivas

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON
